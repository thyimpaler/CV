# UI/UX Build Instructions

## Core Aesthetic
* **Theme:** Ultra-minimalist, high-end developer portfolio (Vercel/Next.js documentation style).
* **Color Palette:** * Background: Pure Black (#000000)
  * Primary Headings: Pure White (#FFFFFF)
  * Body Text: Muted Grey (#888888)
  * Borders/Dividers: Dark Grey (#1A1A1A)
* **Typography:** Clean, modern Sans-Serif (Inter, Geist, or Roboto).
* **Visual Rule:** STRICTLY NO stock photos, 3D illustrations, or colored gradients.

## Layout Rules
1. **Architecture:** Single-page scroll layout.
2. **Component Style:** Use a "Bento Box" grid for the Technical Toolkit section.
3. **Skill Tags:** Display individual bots and chains (e.g., [Carl-bot], [Solana]) as rounded "pills" with a 1px dark grey border.
4. **Buttons & Links:** "Outline" style only. Transparent background, pure white text, 1px solid white border.
5. **Separators:** Use thin `#1A1A1A` horizontal rules to divide experience blocks.
